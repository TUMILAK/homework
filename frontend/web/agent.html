EPUB 汉化

上传未汉化 EPUB：正文按片段批量调用 DeepSeek（JSON 对齐键位，思路参考 AiNiee 等书籍翻译工具），支持正则预处理（送译前）与可选第二遍润色，再写回 XHTML 与 toc.ncx 后重打包至 workspace/epub_out/。Key / 模型来自「设置」。

图片汉化嵌字

上传漫画页或游戏截图：本地 RapidOCR 抽框 → 片段批量走 DeepSeek（与 EPUB 相同的 JSON 批译与术语表）→ 在识别区域内填底并嵌简体中文。流程参考 团子翻译器 的「识图–译–嵌字」思路；服务端需已安装 rapidocr-onnxruntime 等依赖（见仓库 requirements.txt）。输出在 workspace/image_out/。
